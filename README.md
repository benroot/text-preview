# Text Message Preview

A single-page tool for composing and previewing text messages in a phone UI mockup before sending.

## Usage

Type your message in the text area and see it rendered live inside a phone-style chat bubble. Use **Clear** to reset the field or **Copy** to copy the message text.

## Live Site

Hosted via GitHub Pages: `https://benroot.github.io/text-preview`

## Stack

Plain HTML, CSS, and JavaScript — no dependencies, no build step.
